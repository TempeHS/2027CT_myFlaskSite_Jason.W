Project 2

The cipherforge, one of my best inventions, is fully functional! Take a look at how I created this.

Overview

Using the same website as project 1, Mr. Jones (my computer tech teacher), allowed me to freely customise my Cipherforge bot. It sends a normal message such as Hello, and encrypts it to a crazy mixed up messages based on what I have customised.

Lessons

I had to compete 9 lessons in total for this app to fully work. It was a fun time designing this app. I had multiple failure which I fixed from typos or silly mistakes. These lessons were designed to help us create and understand how Ciphering works and what it does.

Ciphering

This might just look like a bunch of words mixed as a string but not, its a code that mixes up words. Lets say you have 'hello' as a word. it shifts the letters by 5, the given number, so the new word would be 'MJQQT'. H + 5 alphabet letters would be M, E + 5 would be J, e.t.c. Block sizes makes it split up, the new word being 'MJQQ T'. Having longer words like Hello World, the block size would split it into little blocks, 'HELL O WO RLD', a space counts as a block too! The noise is a symbol that gets intserted. Given you have '~', you can choose the interval it is put in, so for the code shown, it'll be every 3 characters.

Login

Most people in the class chose to have a login. It allows people to register an account to be able to use the app, such as games like Roblox, Discord, or Instagram. You can create a password with 8 letters or characters, which gets stored into a database (db). This db saves all your login information and allows you to get into your site.

Cipherforge

The best part of them all, the actual ciphering. In the image given, I used the word hello. You are able to change the different settings, at the bottom. This includes the shifting, block size, and more. It allows you to encrypt, but also allows you to decrypt.
